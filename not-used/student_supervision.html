Student Research Projects

I'm interested in supervising theses or research projects at the intersection of natural language processing, machine learning, and cognitive science. Reach out if you're interested in working on these topics.

Possible projects

Interpretability and cognitive plausibility of language models

Probing language models

Research questions: To which extent do computational language models reflect human language processing? Which types of human data can we leverage to increase the interpretability of NLP models?

Evaluation of language representations

CogniVal is an openly available framework for cognitive word embedding evaluation. CogniVal evaluates word embeddings by how well they are able to predict brain activity data of human mental representations of the same words. CogniVal is an established project and there are various possible projects in this area: improving the machine learning methods and user experience, adding more cognitive data sources and language models in multiple languages, etc.

Cognitive and neurolinguistic data resources

In order to use cognitive signals in NLP, we need available datasets of human language processing signals in naturalistic settings. The focus of this research area is to compile and share neurolinguistic corpora in multiple languages tailored for NLP, and to provide tools for the collection and analysis of these data resources.

Multi-modal language modeling

Various data sources recording human language understanding, i.e., eye-tracking and brain activity data recorded during reading text or listening to stories, are available. The goal of this project is to bring these resources together, learn representations for each of them and combine them into a multi-modal language model. The focus of this project is on multi-modal machine learning.

Understanding human labeling behavior

In this collaboration with ETH Zurich, we aim to analyze the labelling process to gain training data for machine learning through crowdsourcing. We are interested in analyzing the relationship between label quality, labelling time, task difficulty, etc. from a machine learning development perspective as well as from a cognitive science perspective.
